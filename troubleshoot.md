# Troubleshooting

## MySQL still doesn't start

Check

```
xampp/mysql/data/mysql_error.log
```

---

## phpMyAdmin keeps loading forever

Run the repair script again as Administrator.

---

## Access Denied

Run the script using

```
Run as Administrator
```

---

## Backup folder missing

The script requires

```
xampp/mysql/backup/mysql
```

which is included with every default XAMPP installation.

---

## Custom XAMPP Location

When prompted, enter your installation path.

Example

```
D:\XAMPP
```

instead of

```
C:\xampp
```
